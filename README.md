# 🛍️ ShopSphere

A full-stack MERN E-Commerce platform built with modern web technologies. ShopSphere provides a complete online shopping experience with authentication, product browsing, shopping cart, wishlist, secure payments, order management, reviews, and an admin dashboard.

---

* User Registration & Login (JWT Authentication)
* Product Search & Filtering
* Product Categories
* Product Details
* Product Reviews & Ratings

* Shopping Cart
* Wishlist
* Checkout
* Razorpay Payment Integration

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Shobhit-Samrat/ShopSphere.git
```

Go into the project

```bash
cd ShopSphere
```

Install frontend dependencies

```bash
cd client
npm install
```

Install backend dependencies

```bash
cd ../server
npm install
```

---

## ▶️ Running the Project

Backend

```bash
cd server
npm run dev
```

Frontend

```bash
cd client
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file inside the `server` folder using the provided `.env.example` template and fill in your own credentials.
